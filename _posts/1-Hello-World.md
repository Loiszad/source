## 1.Hello World ##

```java

public class Hello{

	public static void main(String[] args){

		System.out.println("Hello World!");
	}
}

```

## 2.编译运行Hello World （前提配置好java环境） ##

**①编译命令：javac Hello.java**

**②运行命令：java Hello**
